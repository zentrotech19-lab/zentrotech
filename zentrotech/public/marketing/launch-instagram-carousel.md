## Production Notes

- Export each slide as PNG at 1080x1350 with sRGB color profile
- Use the same Geist font on every slide for consistency
- Maintain a 64px safe-zone padding on every edge so Instagram crop never clips text
- Slide 1 is the only one users see in feed without swiping — must hook in 1.5 seconds
- Slide 10 should mirror slide 1's color palette so the loop feels continuous if reposted as a Reel
